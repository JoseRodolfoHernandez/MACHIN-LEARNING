Logistic Regression - Basic Concepts

What is Logistic Regression?

Logistic regression is a supervised machine learning algorithm used for classification problems. It predicts probabilities instead of continuous values.

Regression vs Classification

Linear regression predicts numeric values, while logistic regression predicts categories.

Binary Classification

It classifies data into two classes (0 or 1).

Target Variable

The output variable is categorical (e.g., Buy or Not Buy).

Features

Input variables such as age, income, visits, etc.

Probability Estimation

Uses a sigmoid function to produce values between 0 and 1.

Decision Boundary

A threshold (usually 0.5) determines the class.

Real Example

Predicting if a customer will purchase a product based on behavior.
