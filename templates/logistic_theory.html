Logistic Regression - Basic Concepts

What is Logistic Regression?

Logistic regression is a supervised machine learning algorithm used for classification problems. It predicts the probability that an instance belongs to a specific class.

Difference Between Regression and Classification

Linear regression predicts continuous values, while logistic regression predicts discrete categories (such as Yes/No or 0/1).

Binary Classification

Logistic regression is commonly used for binary classification, where the output has only two possible values.

Target Variable / Class Label

The target variable represents the output we want to predict (e.g., purchase = 1 or no purchase = 0).

Features / Predictors

These are the input variables used to make predictions, such as age, income, or number of visits.

Probability Estimation

Logistic regression uses a sigmoid function to estimate probabilities between 0 and 1.

Decision Boundary

The decision boundary separates the classes based on probability.

Classification Threshold

A threshold (commonly 0.5) is used to decide the final class: if probability ≥ 0.5 → class 1, otherwise → class 0.

Real-World Example

Predicting whether a customer will buy a product based on their behavior and demographics.
