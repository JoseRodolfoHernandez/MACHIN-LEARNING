Linear Regression - Concepts

What is Linear Regression?

Linear regression is a supervised learning algorithm used to model the relationship between an independent variable (X) and a dependent variable (Y).

Variables

  • Independent Variable (X): Input data
  • Dependent Variable (Y): Output prediction

Training Data

The model is trained using historical data to find patterns between X and Y.

Regression Line

The model finds the best fitting line that represents the relationship between variables.

Equation

Y = mX + b

Example

If a company increases its advertising investment, sales are expected to increase proportionally.
