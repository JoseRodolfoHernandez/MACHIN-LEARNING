Supervised Learning - Linear Regression

What is Supervised Learning?

Supervised learning is a Machine Learning technique where the model learns from labeled data. It uses input-output pairs to identify patterns and make predictions.

What is Linear Regression?

Linear regression is a supervised learning algorithm used to predict continuous values. It models the relationship between an independent variable and a dependent variable using a straight line.

Independent and Dependent Variables

  • Independent Variable (X): Input data (e.g., study hours).
  • Dependent Variable (Y): Output to predict (e.g., grades).

Training Data

Training data consists of historical data used to teach the model the relationship between variables.

Regression Line

The regression line is the best-fit line that represents the relationship between X and Y.

Formula

Y = mX + b

  • m (Slope): Rate of change.
  • b (Intercept): Value when X = 0.

Application Example

Linear regression can be used to predict sales based on advertising investment or time.
