Classification Model - Application

Problem Description

The objective is to classify whether a customer will purchase a product based on their characteristics.

Dataset Explanation

The dataset includes customer information such as age, income, website visits, time spent on site, previous purchases, and discount usage.

Input Variables

  • Age
  • Income
  • Website Visits
  • Time on Site
  • Previous Purchases
  • Discount Used

Target Variable

Purchase decision (0 = No, 1 = Yes)

Data Preparation

The dataset is cleaned, and data is split into training and testing sets.

Model Training

A logistic regression model is trained using historical data.

Prediction Results

The model predicts whether a customer will buy or not based on input values.

Evaluation Metrics

  • Accuracy
  • Precision
  • Recall
  • F1 Score

Interpretation

The model shows how customer behavior influences purchasing decisions. Higher engagement leads to a higher probability of buying.
